Remarks

UnlockRegion(Int64, Int64, LockType) unlocks a region previously locked with the LockRegion(Int64, Int64, LockType) method. Locked regions must later be explicitly unlocked by calling UnlockRegion(Int64, Int64, LockType) with exactly the same values for the libOffset, cb, and dwLockType parameters. The region must be unlocked before the stream is released. Two adjacent regions cannot be locked separately and then unlocked with a single unlock call.

IStreamGetStatistics(StorageStatisticsFlags)
Remarks

GetStatistics(StorageStatisticsFlags) retrieves a reference to the StorageStatistics structure that contains information about this open stream. When this stream is within a structured storage and IStorage::EnumElements is called, it creates an enumerator object with the IEnumSTATSTG interface on it, which can be called to enumerate the storages and streams through the StorageStatistics structures associated with each of them.

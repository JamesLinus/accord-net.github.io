Remarks

The bag-of-words (BoW) model can be used to extract finite length features from otherwise varying length representations. This class can uses any feature detector to determine a coded representation for a given Signal.

For a simpler, non-generic version of the Bag-of-Words model which defaults to the MFCC feature extractor, please see BagOfAudioWords.

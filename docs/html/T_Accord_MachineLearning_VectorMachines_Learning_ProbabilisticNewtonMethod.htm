Remarks

This class implements a L2-regularized L2-loss logistic regression (probabilistic support vector machine) learning algorithm that operates in the primal form of the optimization problem. This method has been based on liblinear's l2r_lr_fun problem specification, optimized using a Trust-region Newton method.
